# 便携 Python + Playwright 批量查价工具

这个工具用于把 Excel 中的发货地址、到货地址和车型列批量提交到货拉拉企业版查价页，并把页面中的 **运费一口价** 和 **总里程** 写回新的 Excel。

## 网页入口

推荐在本机双击工具目录里的：

```text
打开查价工具网页.hta
```

它会自动启动本地服务，并打开查价控制页。

如果上面的入口打不开，再双击备用启动脚本：

```text
启动查价工具.bat
```

服务启动后，可以点击下面这个链接进入控制页：

[打开批量查价工具网页](http://127.0.0.1:18765/)

这个链接是本地网页入口，不是公网网站。每个人都需要在自己的电脑上先启动工具服务，链接才会打开他自己电脑上的查价工具。

## 使用方式

1. 下载或复制完整工具文件夹到本机，并解压到普通目录，例如桌面或 `D:\批量查价工具`。
2. 双击 `打开查价工具网页.hta`。
3. 如果没有自动打开控制页，再双击 `启动查价工具.bat`，然后回到 README 点击上面的“打开批量查价工具网页”链接。
4. 在控制页里点击“选择 Excel 文件”，选择一个或多个要查价的 `.xlsx`。
5. 输入或保留货拉拉页面 URL：
   `https://bfe-epc-web-v.huolala.cn/#/order-center/same-city?from=menu`
6. 点击“打开专用 Edge”，在打开的 Edge 中完成登录或验证码。（需人工先登录）
7. 回到控制页点击“用所选 Excel 创建任务”，再点击“开始查价”。
8. 结果文件会生成在工具目录下的 `output\<时间戳>\`。
9. 运行结束后专用 Edge 会停留在最后一次查价页面，方便核对页面价格和 Excel 记录。

如果要批量固定处理某个目录，也可以把 Excel 放到工具目录下的 `input` 文件夹，然后点击“使用 input 目录（备用）”。结果同样生成在工具目录下的 `output\<时间戳>\`。

## 另一台电脑打不开怎么办

- 如果是在 GitHub 网页里点 `启动查价工具.bat`，只会看到代码页，不会启动程序。必须先把整个项目下载成 ZIP，解压后在 Windows 文件夹里双击本地文件。
- 普通用户优先双击 `打开查价工具网页.hta`，不要双击 README 里的链接本身。
- 如果双击 `.bat` 什么都没有，说明当前文件夹可能不是完整便携包，或那台电脑没有 Python 运行环境。请使用包含 `runtime\python\python.exe` 的便携版文件夹，或先安装依赖。
- 如果 `.bat` 被当成文本打开，说明 Windows 文件关联异常。可以右键 `启动查价工具.bat`，选择“打开方式”，使用“Windows 命令处理程序”；也可以在地址栏输入 `cmd` 打开命令行后运行：

```bat
启动查价工具.bat
```

- 启动成功后，浏览器访问的是本机地址 `http://127.0.0.1:18765/`。这个地址只对当前电脑有效，不能从另一台电脑直接打开。

## Excel 约定

- 默认读取第一个 Sheet。
- 第 2 行必须包含 `供应商名称`、`发货地址`、`距离`、`到货地址`。
- 第 1 行和第 2 行共同定义价格列，例如：
  - 第 1 行：`厢式货车&飞翼门`
  - 第 2 行：`9.6`、`12.5&13`、`16.5&17.5`
- 工具只填空白的距离和价格单元格，已有值不会覆盖。若想覆盖已有值并生成新值，需勾选覆盖选项。
- 原始 Excel 不会修改，每个输入文件都会生成同名 `_查价结果.xlsx` 副本。

## 货拉拉网页读取规则

- 用车时间和“快车”保持页面现状不变。
- 价格读取页面中的 `运费一口价`，不会读取已优惠后的 `总计`。
- 路程读取页面中的 `总里程...公里`。
- 车型长度默认映射：
  - `9.6` -> `9米6`
  - `12.5&13` -> `13米`
  - `16.5&17.5` -> `17米5`
  - ...

如果货拉拉页面控件 DOM 有变化，优先调整 `configs/site.huolala.json`。

## 开发验证

核心 Excel 测试不依赖 Playwright：

```powershell
python -m unittest discover -s tests
```

完整网页自动化需要安装依赖后运行：

```powershell
python -m pip install -r requirements.txt
python -m playwright install msedge
```
