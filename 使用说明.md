# 货拉拉报价工具使用说明

## 首次使用

1. 打开 `启动货拉拉报价工具.bat`，或直接打开 `HuolalaQuoteTool.exe`。
2. 点击 `选择 Excel`，选择要报价的 `.xlsx` 文件。
3. 在 `字段识别预览` 里确认 `供应商名称`、`发货地址`、`到货地址`、`距离` 和车型价格列识别正确。
4. 选择复跑策略：
   - `只补空白价格`：只处理空白价格，不覆盖已有报价。
   - `全部重跑`：重新读取距离和所有匹配车型报价。
   - `只重跑失败项`：只处理带失败批注的单元格。
5. 点击 `开始`。
6. 浏览器打开货拉拉企业版同城下单页后，人工登录账号。程序不会保存账号密码。
7. 登录并停留在同城下单页后，回到工具点击确认，程序开始批量查询。

## 结果文件

工具不会覆盖原 Excel，会在原文件同目录生成：

```text
原文件名_货拉拉报价结果_YYYYMMDD_HHMMSS.xlsx
```

程序每处理完一行都会保存一次。失败原因会写入对应单元格批注。

## 车型规则表

规则表在 `config/vehicle_rules.csv`，可以用 Excel 打开编辑。字段含义：

- `Excel表头关键词`：Excel 里车型报价列的表头关键词，多个写法用英文分号 `;` 分隔。
- `网页车长`：货拉拉网页上要选择的车长，例如 `9米6`。
- `网页车型要求`：货拉拉网页上要选择的车型要求，多个要求用英文分号 `;` 分隔。
- `启用`：填 `是` 启用，填 `否` 跳过。

只有匹配到启用规则的车型价格列会被处理，其他列保留不动。

## 打包

在开发电脑上运行：

```powershell
python -m pip install -r requirements.txt
python scripts\build_windows_portable.py
```

生成目录：

```text
dist\HuolalaQuoteToolPortable
```

复制整个文件夹到另一台 Windows 电脑即可运行。工具会优先调用系统自带 Microsoft Edge。
